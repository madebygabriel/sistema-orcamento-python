# Descrição do Projeto 📝
- Um cliente pode ter vários orçamentos, porém um orçamento é único para cada cliente;
- Sistema desenvolvido para praticar boas práticas de programação, classes, funções e relacionamento entre classes;
- Sistema simples, porém é objetivo e com potencial de engajar em um sistema empresarial de orçamentos;
- Futuramente o objetivo é integrar preço da mão de obra, Herança, Banco de Dados, API e interface gráfica, para assim uma melhor visão do sistema.

# Status do Projeto ✅
🚧 Projeto em construção 🚧

# Funcionalidades 🔨
- Criação de: clientes, produtos e orçamentos;
- Inserção de produtos;
- Listagem de produtos do orçamento;
- Salvar o orçamento em .json, com conteúdos: informações do cliente, informações de TODOS os produtos, além do preço total do orçamento.
